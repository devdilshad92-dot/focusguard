# Assets

FocusGuard uses **symbolic icons from the system icon theme** at runtime
(`alarm-symbolic`, `cup-symbolic`, `focus-windows-symbolic`, …), so no bitmap
artwork needs to ship for the extension to work.

Put store/marketing artwork here:

- `screenshot-panel.png` — the panel indicator + open menu (referenced by README)
- `screenshot-overlay.png` — the fullscreen break overlay
- `screenshot-prefs.png` — the preferences window
- `icon.svg` — optional project logo for the repo / EGO listing

Recommended screenshot size for extensions.gnome.org: at least 1280px wide, PNG.
